=# Routing Advanced

A React.js project built to explore advanced routing concepts using React Router DOM. The application demonstrates nested routes, dynamic route parameters, navigation handling, and reusable UI components styled with Tailwind CSS.

## Features

- Client-side routing with React Router DOM
- Nested Routes
- Dynamic Route Parameters
- Navigation Bar
- Course Listing and Details Pages
- Product and Category Pages
- Custom 404 Not Found Page
- Responsive UI with Tailwind CSS
- Reusable React Components

## Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
- HTML5

## Installation & Setup

```bash
git clone <repository-url>
cd Routing-advanced
npm install
npm run dev
```

## Learning Objectives

- React Router DOM
- Nested Routing
- Dynamic Routing
- Route Parameters
- Component-Based Architecture
- Responsive Design with Tailwind CSS
